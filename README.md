# MinimarketApp

## 📌 Descripción
MinimarketApp es una aplicación web básica diseñada como proyecto para la clase de Diseño Web. Su objetivo es simular una página moderna y responsiva para un minimercado, con enfoque en usabilidad y diseño visual.

## 🚀 Propósito del proyecto
Este proyecto fue desarrollado con fines educativos. Su propósito es:
- Aplicar conocimientos básicos de HTML, CSS y JavaScript.
- Desarrollar una landing page sencilla.
- Enlazar un proyecto con un repositorio en GitHub.

## 🛠 Tecnologías utilizadas
- **HTML5**: Estructura del sitio.
- **CSS3**: Estilos personalizados y diseño responsivo.
- **JavaScript**: Interactividad básica con eventos (como alertas).
